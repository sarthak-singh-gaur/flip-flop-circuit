What is a Flip-Flop?

üîÑ Sequential Logic

A flip-flop is a fundamental digital circuit that serves as a single-bit memory element. It maintains one of two stable states (0 or 1) until changed by an input.

‚ö° Memory Element

Unlike combinational circuits which produce outputs solely from current inputs, flip-flops introduce memory by depending on past states.

üéØ Clocked Operation

Flip-flops synchronize with clock signals, triggering state changes at precise edges (rising/falling), enabling predictable behavior in synchronous systems.

Master-Slave Flip-Flop

The master-slave flip-flop consists of two latches in series. The master captures input when clock is low, and the slave transfers to output when clock is high. This prevents race conditions.

Setup Time (tsu)

Minimum time input must be stable before clock edge. Ensures proper capture of input data.

Hold Time (th)

Minimum time input must remain stable after clock edge. Prevents premature change.

Propagation Delay (tpd)

Time from clock edge to output stable. Limits maximum clock frequency.

fmax = 1 / (tpd + tsu + th)

Metastability

When setup/hold times are violated, the output may oscillate before settling. This is called metastability.

Solution: Use synchronizers (2-3 flip-flops in series) to increase MTBF (Mean Time Between Failures).
